## Projects

### NGO Management System

> 完整的非營利組織管理系統，包含員工後台與用戶前台，獨立設計、開發、部署。

- 用戶前台：ASP.NET MVC + Bootstrap + jQuery（RWD）
- 員工後台：React 18 + Vite + MUI
- API：ASP.NET Core 9 Web API · SQL Server
- 部署：Cloudflare Tunnel（24/7 live）
- ECPay 金流、OpenAI 整合（GPT / DALL-E / Whisper）、Google OAuth

### Trail Guide

> 台灣步道資訊平台，支援搜尋、收藏、評論、GPS 附近步道，具備 PWA 可安裝至手機桌面。

- 前端：React 19 + TypeScript + Vite + MUI v7
- 後端：ASP.NET Core 8 Web API · PostgreSQL
- JWT + Google OAuth · Leaflet 地圖 · Cloudinary 圖片上傳
