## 🚀 Projects

### 🏢 NGO Management System　恩舉非營利組織管理平台

> 完整的非營利組織管理系統，包含員工後台與用戶前台，獨立設計、開發、部署。

**架構：**
- 用戶前台：ASP.NET MVC + Bootstrap + jQuery（RWD）
- 員工後台：React 18 + Vite + MUI
- API：ASP.NET Core 9 Web API
- 資料庫：SQL Server
- 部署：Cloudflare Tunnel（24/7 live）

**功能亮點：**
- ECPay 金流串接（捐款 / 物資購買）
- OpenAI 整合（GPT 文案優化、DALL-E 圖片生成、Whisper 語音轉文字）
- Google OAuth 登入
- 成就系統、活動報名、物資管理

### 🥾 Trail Guide　步道導覽平台

> 台灣步道資訊平台，支援搜尋、收藏、評論、GPS 附近步道，具備 PWA 可安裝至手機桌面。

**架構：**
- 前端：React 19 + TypeScript + Vite + MUI v7
- 後端：ASP.NET Core 8 Web API
- 資料庫：PostgreSQL
- 認證：JWT + Google OAuth
- 地圖：Leaflet + OpenStreetMap

**功能亮點：**
- 步道搜尋（關鍵字、縣市、難度）
- GPS 附近步道
- 評論 + 圖片上傳（Cloudinary）
- 口袋名單、已去過標記
- PWA 支援（可安裝至手機）
